Import produse (fără terminal)

Lipește CSV sau încarcă fișier. Coloane: category,name,finish,material,price,old_price,opt_usd,feat,image,badge. Categorii: smesitel, unitaz, rakovina, polotenec, dush, kuhnya.

Цена в lei. Если price пусто — считается из opt_usd × 80. Если фото нет — подставится плейсхолдер-hero. После импорта перейди в «Export» и обнови сайт.

Catalog

Редактируй цену, бейдж, скрывай товары. Изменения попадут в Export.

Pricing engine

Корректировка цен по всему каталогу или категории.

Напр. «+10» поднимет цены на 10%, «-5» снизит на 5%. Старые цены (old_price) пересчитываются пропорционально.

Sistem de tendințe (Trend Score)

Trend Score = preț mic + reducere + rating + recenzii + badge. Top товары для рекламы.

Export & Publicare pe site

Сгенерируй обновлённый data.js и опубликуй — без терминала.

  1. Нажми «Descarcă data.js» — скачается файл.
  2. Открой репозиторий на GitHub → папка assets/js/ → Add file → Upload files → загрузи новый data.js (заменит старый) → Commit changes.
  3. Через ~1 минуту сайт обновится. Терминал не нужен.
Альтернатива (быстрее): положи скачанный data.js в ~/Downloads/aqualux-md/assets/js/ и выполни git add -A && git commit -m update && git push.
